# 📦 Project Overview

Repository ini berisi implementasi model klasifikasi citra untuk pemilahan sampah menggunakan Convolutional Neural Network (CNN).
* **`Jurnal`**
https://j-ptiik.ub.ac.id/index.php/j-ptiik/article/view/15793
* **`Dataset`**
https://drive.google.com/drive/folders/1zbqEWpVDE091mvFDALJr-C33Ymf_84uF?usp=sharing

## 📂 Struktur File

* **`min2.ipynb`**
  Berisi kode utama untuk proses pelatihan model (training), termasuk preprocessing, augmentasi, dan konfigurasi model.

* **`kfold.ipynb`**
  Berisi kode untuk evaluasi model menggunakan metode **K-Fold Cross Validation** guna mengukur performa dan stabilitas model.

* **`test.ipynb`**
  Digunakan untuk melakukan pengujian (inference) terhadap gambar baru menggunakan model yang sudah dilatih.

* **`test.jpg`**
  Contoh gambar yang digunakan untuk proses testing model.

## 🚀 Tujuan

Project ini bertujuan untuk mengklasifikasikan citra sampah ke dalam beberapa kategori menggunakan pendekatan deep learning.
